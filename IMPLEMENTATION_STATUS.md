## 🔧 データベースマイグレーション

評価システムを有効にするには、以下のSQLをSupabase Dashboardで実行してください：

```bash
# Supabase Dashboard → SQL Editor で実行
scripts/migrate-add-ratings-system.sql
```

このSQLは以下を実行します：
- `ratings`テーブルの作成
- `companies`, `organizations`, `students`テーブルに評価カラム追加
- 評価平均値自動更新トリガーの作成
